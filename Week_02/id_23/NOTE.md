# 学习笔记

本周的重点算法是跳表、散列表、哈希算法、二叉树、红黑树。接下来分别用以两句话描述一下我对该算法的理解。

## 跳表

- 为有序链表建立多级索引。
- 查询时间复杂度 从 O(n) 到 O(log n)；
- 索引的存储空间复杂度 O(n)
- 用途主要应用于 redis 的 order set 的实现。

## 散列表

- 查询时间复杂度 O(1)

## 哈希算法

- 主要解决 hash 冲突

## 二叉树

- 一种特殊的树结构
- 算法实现一般会用到递归

## 红黑树

- 一个二叉搜索树的查询平均时间复杂度为 O(log n) ；但是当树极度不平衡的时候 时间复杂度退化到 O(n)
- 红黑树 主要保证树的平衡。
