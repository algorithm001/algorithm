# 学习笔记

- 动态规划
  - 自底向上
  - 通过求子问题的最优解， 可以获得原问题的最优解
  - 有重叠子问题 + 最优子结构，还必须记忆搜索，即可用动态规划

## 常用数据结构的小结及 ADT API
- Trie 字典树, 每个节点都是 map, 根节点为空
  - add(val)
  - isWord(val)
  - contains(val)
  - isPrefix(val)

## 其他
- JS 初始化二维数组 `Array(m).fill().map(() => Array(n).fill(-1))`
